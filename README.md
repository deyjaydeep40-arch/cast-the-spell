# ✨ Cast The Spell - Interactive Game

An enchanting interactive game where you cast magical spells, solve puzzles, and navigate through mystical worlds. Built with engaging gameplay mechanics and stunning visual effects.

## ✨ Key Features

- 🪄 **Spell Casting** - Learn and cast magical spells
- 🧩 **Puzzles** - Brain-teasing challenges to overcome
- 🌍 **Multiple Worlds** - Explore different magical realms
- 👹 **Enemies** - Battle magical creatures
- 💎 **Collectibles** - Gather treasures and power-ups
- 🎨 **Visual Effects** - Beautiful spell animations
- 🎮 **Game Progression** - Level-up system
- 📱 **Responsive Design** - Play on any device

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3 (animations & effects)
- JavaScript
- Canvas/WebGL for graphics

**Game Engine:**
- Custom game loop
- Physics simulation
- Collision detection
- Event handling

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- No installation required

### Installation

```bash
# Clone the repository
git clone https://github.com/deyjaydeep40-arch/cast-the-spell.git
cd cast-the-spell

# Open in browser
open index.html

# Or use a local server
python -m http.server 8000
# Visit http://localhost:8000
```

## 🎮 How to Play

### Basic Controls
- **Arrow Keys** - Move character
- **Space** - Jump
- **Mouse Click** - Cast spell
- **Keys 1-5** - Select spell
- **P** - Pause game
- **R** - Restart level

### Spell System

#### Fire Spell 🔥
- Damage: 10
- Range: Medium
- Cooldown: 1 second
- Cost: 10 mana

#### Ice Spell ❄️
- Damage: 8
- Range: Short
- Cooldown: 2 seconds
- Cost: 15 mana
- Special: Freezes enemies

#### Lightning Spell ⚡
- Damage: 15
- Range: Long
- Cooldown: 3 seconds
- Cost: 20 mana

#### Healing Spell 💚
- Heal: 20 HP
- Range: Self
- Cooldown: 5 seconds
- Cost: 25 mana

#### Shield Spell 🛡️
- Duration: 5 seconds
- Protection: 50%
- Cooldown: 4 seconds
- Cost: 15 mana
